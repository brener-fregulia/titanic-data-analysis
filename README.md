# Desafio de Análise de Dados – Titanic (CSV)

## Sobre o projeto:
Exercício prático de Análise de Dados utilizando apenas Python puro e bibliotecas essenciais (`pandas`, `numpy`, `matplotlib`).

---
## Objetivo:
- Ler o dataset do Titanic
- Responder 10 perguntas analíticas apenas com código
- Produzir ao menos um gráfico de visualização
- Organizar as respostas em scripts separados para cada pergunta

---
## Dataset:
- Nome: **titanic.csv**
- Fonte: [Titanic Dataset - GitHub](https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv)

---
## Estrutura de Pastas:
Desafio-CSV-Titanic/
├── data/
│ └── titanic.csv
├── src/
│ ├── Q01.py → Quantas linhas e colunas tem o dataset?
│ ├── Q02.py → Quantas pessoas sobreviveram?
│ └── ... até Q10.py
├── README.md
└── requirements.txt

---
## Como executar:

1. Clone o repositório
2. Crie um ambiente virtual (opcional)
3. Instale as dependências: pip install -r requirements.txt
4. Execute cada script dentro da pasta 'src/'

## Perguntas respondidas:
✅ Total de linhas e colunas  
✅ Total de sobreviventes e não-sobreviventes  
✅ Idade média geral e segmentada  
✅ Tarifa média por classe  
✅ Correlação entre idade e tarifa  
✅ Gráfico de dispersão  
✅ Contagem de homens e mulheres  
✅ Número de crianças sobreviventes
